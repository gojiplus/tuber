tuber: Access YouTube via R

Install, Load the package

To install the latest version from CRAN:

install.packages("tuber")

The latest development version of the package will always be on GitHub. Instructions for installing the package from Github are provided below.

# install.packages('devtools')
devtools::install_github("soodoku/tuber", build_vignettes = TRUE)

Next, load the package:

library(tuber)

Using the package

To get going, get the application id and password from Google Developer Console (see https://developers.google.com/youtube/v3/getting-started). Enable YouTube APIs. Then set the application id and password via the yt_oauth function. For more information about YouTube OAuth, see YouTube OAuth Guide.

yt_oauth("998136489867-5t3tq1g7hbovoj46dreqd6k5kd35ctjn.apps.googleusercontent.com", "MbOSt6cQhhFkwETXKur-L9rN")

# get stats:
res <- data.frame()

    temp <- get_stats(i)
    temp$vid_id <- i
    res  <- rbind(res, temp)
}
