# DELETE

DELETE

## Usage

``` r
tuber_DELETE(path, query, auth = "token", ...)
```

## Arguments

- path:

  path to specific API request URL

- query:

  query list

- auth:

  A character vector of the authentication method, either "token" (the
  default) or "key"

- ...:

  Additional arguments passed to
  [`DELETE`](https://httr.r-lib.org/reference/DELETE.html).

## Value

list
