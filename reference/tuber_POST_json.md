# POST encoded in json

POST encoded in json

## Usage

``` r
tuber_POST_json(path, query, body = "", ...)
```

## Arguments

- path:

  path to specific API request URL

- query:

  query list

- body:

  passing image through body

- ...:

  Additional arguments passed to
  [`GET`](https://httr.r-lib.org/reference/GET.html).

## Value

list
