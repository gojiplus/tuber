# PUT

PUT

## Usage

``` r
tuber_PUT(path, query, body = "", auth = "token", ...)
```

## Arguments

- path:

  path to specific API request URL

- query:

  query list

- body:

  JSON body content for the PUT request

- auth:

  A character vector of the authentication method, either "token" (the
  default) or "key"

- ...:

  Additional arguments passed to
  [`PUT`](https://httr.r-lib.org/reference/PUT.html).

## Value

list
